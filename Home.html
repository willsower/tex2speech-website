THE PROJECT, IN SIMPLE

The Tex2Speech application can be run locally on any machine. Users are able to upload their LaTeX documents, input documents, .bib files, or even .zip/.tar files and get listenable audio in .mp3 file format. User's can even upload multiple files during the same sythesis process. To do this, we are using Amazon Polly as the speech sytehsizer and will convert the LaTeX document int oa marked up Speech Sythesis Markup Language (SSML) document. SSML allows us to control how words are pronounced, break time, and pauses throughout the document.
